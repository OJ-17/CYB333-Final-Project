# CYB333 Final Project

## Project Description
This project was created for CYB333 Security Automation. The purpose of this project is to show how Python can be used to automate cybersecurity event detection.

## What the Program Does
This Python program detects suspicious security events from log data, including failed logins, successful logins after failed attempts, new user creation, and suspicious PowerShell activity.

## Tools Used
- Python
- Visual Studio Code
- GitHub
